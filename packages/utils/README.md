# @g14o/utils

> **Deprecated.** This package is a compatibility shim. Install and import from [`@g14o/core`](../core/README.md) instead.

This shim is maintained in the monorepo for migration only and is **not published** to npm.

## Migration

| Old import | New import |
|------------|------------|
| `@g14o/utils` | `@g14o/core` |
| `@g14o/utils/types` | `@g14o/core/types` |
| `@g14o/utils/config` | `@g14o/core/config` |

```bash
pnpm remove @g14o/utils
pnpm add @g14o/core
```

See the [@g14o/core README](../core/README.md) for full documentation.
