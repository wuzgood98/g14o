# @g14o/ratelimit

> **Deprecated.** This package is a compatibility shim. Install and import from [`@g14o/core/ratelimit`](../core/README.md) instead.

This shim is maintained in the monorepo for migration only and is **not published** to npm.

## Migration

| Old import | New import |
|------------|------------|
| `@g14o/ratelimit` | `@g14o/core/ratelimit` |

```bash
pnpm remove @g14o/ratelimit @g14o/utils
pnpm add @g14o/core
```

See the [@g14o/core README](../core/README.md) for full documentation.
