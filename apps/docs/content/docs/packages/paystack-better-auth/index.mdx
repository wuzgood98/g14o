Paystack billing plugin for [Better Auth](https://better-auth.com). Supports customers, hosted checkout, subscriptions, authorization charges, and webhook synchronization.

Requires [@g14o/paystack](/packages/paystack) for the Paystack API client.

## Installation

```bash
pnpm add @g14o/paystack @g14o/paystack-better-auth better-auth zod
```

## What it provides

- Paystack customer sync on sign-up
- Hosted checkout for one-time payments
- Subscription upgrades, cancel, resume, and listing
- Webhook verification and event deduplication
- Database schema for subscriptions, payments, and webhook events
